Identidad Visual

MiLuca

Tu coach financiero con IA. MiLuca rompe con la estética bancaria tradicional para proponer una identidad digital, vibrante y accesible. El sistema visual traduce las finanzas en una experiencia "cero complicaciones", utilizando un lenguaje gráfico diseñado específicamente para generar confianza en la Generación Z.

Identidad central. El logotipo combina solidez y dinamismo, adaptándose perfectamente a interfaces móviles como icono de aplicación.
Sistema cromático digital. Se prioriza el "Azul ML" y el "Violeta ML" para crear una paleta eléctrica y tech, que destaca en pantallas y modo oscuro.
Tipografía y jerarquía. Uso de Hurme Geometric Sans para titulares con personalidad moderna, complementada con Inter para garantizar máxima legibilidad en datos financieros.
Iconografía "Soft". Un set de iconos de trazo grueso y terminaciones curvas que suaviza la interacción con conceptos técnicos, haciendo la navegación amigable.
